Щоб запустити програму потрібно зайти в import_from_txt-bin-debug. Запускаємо import_from_txt.exe.
Програма дозволяє нам відкривати текстові файли, для збереження усіх в один результуючий файл.
Також дозволяє проводити розрахунки для обрахування дисперсії і середньоквадратичного відхилення.

Кнопка open file дозволяє відкривати .txt файл по одному. Кнопка Open directory дозволяє відрити директорію 
в якій знаходяться усі файли, які потрібно об'єднати в один .txt файл.
Summary кнопкаа для розраунків дисперсії і середньоквадратичного відхичення,  відмітка wighted дозволяє 
провести розрахунки з коефіцієнтами зважуваннями. Save as зберігає файл з усіма даними, а також включає 
розрахунки.
Clean Save зберігає лише дані для подальшого розрахунку в інших програмах.
